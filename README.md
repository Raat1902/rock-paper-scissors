# Rock Paper Scissors (Java)

A simple terminal-based **Rock, Paper, Scissors** game written in **Java**.

## Features
- Play against a computer opponent
- Random computer choices
- Score tracking (player vs computer)
- Tracks games played
- Shows your win rate (%) after each round
- Input validation
- Play as many rounds as you want (quit anytime)
